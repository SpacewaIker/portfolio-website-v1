Stagiaire en développement géomatique

Mai 2022 - Juil 2022
  • JavaScript
  • Node.js
  • Docker
  • PostgreSQL
  • Preuve de concept
  • A développé avec succès une preuve de concept pour une application web de routage en intérieur avec JavaScript/Node.js.
  • A connecté une application web front-end avec une base de données PostgreSQL roulant sur un container Docker Ubuntu.

Pour ce stage, j'ai été chargé de développer une preuve de concept pour une application web liée au routage en intérieur. Pendant le développement, j'ai du rechercher les technologies possibles et faire des décisions sur quelles technologies seraient les plus adaptées au projet. J'ai aussi du m'adapter à cause de problèmes de compatibilité que j'ai rencontré ce qui m'a poussé à utiliser des containers Docker pour le back-end de l'application.

L'application a été développée en JavaScript avec Yarn et Webpack. Pour le back-end, J'ai utilisé Docker Compose pour créer une base de données PostgreSQL, un API PostgREST et un engin de routage qui travailleraient tous les trois ensemble. L'appliation web a été deployée sur un serveur IIS.

À la fin du stage, l'application était totalement fonctionelle et répondait aux exigences du client.
