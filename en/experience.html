For my second internship, I have the chance to work as a research assistant at the Reliable Silicon Systems Lab at McGill University. My task was to set up a machine learning development board and to figure out how to get a computer vision model to run on it. The development board was a Raspberry Pi with a camera module with a hardware acceleration chip for machine learning tasks.

During the internship, I implemented multiple computer vision models from published papers, such as LeNet and PortraitNet, using Tensorflow. Each model had to be quantized to 8-bit integers using Tensorflow Lite, before being processed and pacakged with the board's proprietary toolchain. Then, a C++ application would use the packaged model to run directly on the edge device.

By the end, I had successfully implemented and run two computer vision models and a third one was in progress.

For this internship I was tasked with developing a proof-of-concept for a web application related to indoor routing. During the development, I had to research the available technologies and make decisions as to which technologies were best suited for this project. I also had to adapt to some compatibility issues I encountered and instead opted for Docker containers for the back-end of the application.

The application was developed in JavaScript with Yarn and Webpack. For the back-end, I used Docker Compose to create a PostgreSQL database, a PostgREST API, and a routing engine that would all three work together. The web application was deployed on an IIS server.

At the end of the internship, the application was fully functional and responded to the clients' requirements.
